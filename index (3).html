Connect your Google Sheet

This dashboard reads its project list from a published Google Sheet, so anyone on the team can add or edit projects without touching code.

  1. In Google Sheets: File → Share → Publish to web
  2. Choose the correct tab, set format to Comma-separated values (.csv), click Publish
  3. Copy the generated link and paste it below
